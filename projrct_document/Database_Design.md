# **データベース設計**

# 説明
* amac(AnonymousMACAdress)

  ハッシュ化されたMACアドレス。一人が複数台の端末を持っていることを想定している。

* id

  学籍番号。先頭小文字のk00000というフォーマットで保存。

* name

  その人の名前。

* entrytime

  入ってきた時間。

* updatetime

  直近で在室が確認できた時間

* exittime

  最後に在室していたであろう時間。

* room

  部屋番号。
